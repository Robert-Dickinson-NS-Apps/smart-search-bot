# Smart Search Bot

Fast, structured discovery across technical and domain content.

## About

Smart Search Bot is a search-focused web app built for quickly finding and organizing technical and domain-specific content. Rather than returning a flat list of results, it emphasizes structure, so users can move from a search query to the right source with less digging.

## What's Inside

The app includes a citation map experience for navigating and understanding how search results relate to their underlying sources. That makes it better suited to research and technical workflows, where knowing where an answer comes from matters as much as the answer itself, than a plain keyword search box.

## Tech Stack

Built with TypeScript, Vite, and Bun, using a Lovable-managed project structure, with ESLint and Prettier for code quality.

## Getting Started

Clone the repository and install dependencies with Bun.

git clone https://github.com/Robert-Dickinson-NS-Apps/smart-search-bot.git
cd smart-search-bot
bun install
bun run dev
